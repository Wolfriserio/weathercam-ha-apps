# Weathercam Calendar

Ingress UI for browsing daily timelapse videos by year and month.

Expected structure:

- `/media/weathercam/timelapse/YYYY/MM/DD.mp4`
- optional thumbnails: `/media/weathercam/timelapse/YYYY/MM/DD.jpg`

Features:

- year/month picker
- only available days are clickable
- popup HTML5 player
- download button
- latest available day shortcut
- thumbnail backgrounds on existing days
